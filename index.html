Colored Pipes

The objective of this game is to connect the pairs of colored pipes from one starting point to the other. The pipes are not allowed to intersect one of another color. Doing so will overwrite that drawn pipe. Find a way to build around so that each pipe is connected and that there are no gray spaces. Only one click/hold is allowed per color. A second click will reset the pipe! Each color needs a pipe to be created!
Select a level to begin. Good luck!
